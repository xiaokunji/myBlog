.ignore#

git的忽略文件, 可以帮我们添加常见的忽略文件

.ignore - IntelliJ IDEs Plugin | 插件地址

Custom Postfix Templates#

提供常见的后缀模板 快捷方式

idea自带一些, 比如 .soutv 会打印当前变量

该插件提供更多的方式, 比如字符串的判空

该插件默认每天更新一次, 这会导致idea卡顿, 建议取消该选项

该插件也挺吃内存的

Custom Postfix Templates - IntelliJ IDEs Plugin | 插件地址

Easy Javadoc#

快捷生成文档注释, 支持类/ 方法 / 变量

光标点中方法/变量/类, 按 快捷键: ctrl + \

不是选中, 是光标落在上面即可

若选中, 则会翻译

Easy Javadoc - IntelliJ IDEs Plugin | 插件地址

Grep Console#

控制台的优化展示

不同级别的日志展示不同的颜色; 可以按关键词过滤日志

Grep Console - IntelliJ IDEs Plugin | 插件地址

Maven Helper#

pom文件优化展示

梳理pom文件的依赖情况

方便找有哪些包, 以及包的版本, 以及包的冲突情况

Maven Helper - IntelliJ IDEA & Aqua Plugin | 插件地址

MyBatisX#

mybatis的提效插件, 能从mapper的接口层文件直接跳到mapper的sql层

也支持表级别的代码生成器( 可以生成 control / service / &mldr;/ dao / mapper等等文件)

MyBatisX - IntelliJ IDEA Plugin | 插件地址

RestfulTool#

接口地址的搜索插件

方便搜索代码中的接口地址

快捷键: ctrl + alt + /

RestfulTool - IntelliJ IDEs Plugin | 插件地址

String Manipulation#

字符串操作插件

支持字符串之前的各种转换, 也还有其他的一些功能

例如: 驼峰转下划线 ; 首字母大写 ; 驼峰转空格等等

默认不提供快捷键, 可以自己设置

String Manipulation - IntelliJ IDEs Plugin | 插件地址

Translation#

翻译软件, 支持各种翻译引擎

自从谷歌翻译不能用之后, 翻译就不好用了, 不是插件的问题, 是翻译引擎的问题

插件官方推荐使用阿里, 百度等国产免费的引擎

选中单词后, 快捷键: ctrl + shift + y, 翻译单词

快捷键: ctrl + shift + o 打开翻译窗口

翻译源码的注释(当看源码的注释时, 自动翻译):

形如:

设置路径: setting > Tools >Translation

右下角可以切换引擎

Translation - IntelliJ IDEs Plugin | 插件地址
